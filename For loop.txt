• We can generate a sequence of numbers using
range() function. range(10) will generate numbers
from 0 to 9- 10 numbers).
• In range function we have to define the start, stop
and step size as range(start,stop,step size). step 
size defaults to 1 if not provided.
Syntax
for var in range (start, stop, steps):
body of the loop
